## 概述

訂位模組的核心資料結構。所有卡片元件都消費同一個 `ReservationData` 介面。

## ReservationData 欄位清單

### 訂位基本資訊

| 欄位 | 型別 | 說明 | 顯示於 |
|------|------|------|--------|
| `uuid` | string | 訂位唯一識別碼 | 內部使用 |
| `reservationId` | string | 訂位編號（人類可讀） | ReviewReserveCard |
| `rsptCod` | string? | 廳別代碼 | 內部使用 |
| `reservationDate` | string | 訂位日期 | ReviewReserveCard, PreviewReserveCard |
| `startTime` | string | 開始時間 | 全部卡片 |
| `endTime` | string | 結束時間 | TimelineViewCard, 過期判定 |
| `mtimeCod` | string | 餐期代碼 | 內部使用 |

### 狀態欄位

| 欄位 | 型別 | 說明 | 詳見 |
|------|------|------|------|
| `reservationType` | RSV/WLB/WIO/WLK | 訂位類型 | [[狀態系統#訂位類型]] |
| `reservationStatus` | CNF/CXL/WTC/WIO/WLB | 訂位狀態 | [[狀態系統#訂位狀態]] |
| `reminderStatus` | NNF/NDF/NTF/HLD | 通知狀態 | [[狀態系統#通知狀態]] |
| `reservationSource` | ONL/TEL/WLK/STF/CRS/PMS/POS | 訂位來源 | [[狀態系統#訂位來源]] |
| `cancelSource` | CCL/SCL/SYS/PMS? | 取消來源 | — |
| `cancelReason` | NSW/OTH/NDC/WTO? | 取消原因 | [[狀態系統#取消原因]] |

### 人數

| 欄位 | 型別 | 說明 |
|------|------|------|
| `numberOfAdults` | number | 大人數 |
| `numberOfChildren` | number | 小孩數 |
| `numberOfInfants` | number | 嬰兒數 |

### 客戶資訊 (customer)

| 欄位 | 型別 | 說明 | 顯示於 |
|------|------|------|--------|
| `fullName` | string? | 客戶姓名 | 全部卡片 |
| `firstName` | string | 名 | CreateReserveCard |
| `lastName` | string | 姓 | CreateReserveCard |
| `gender` | M/F/U/S? | 性別 | 全部卡片（icon） |
| `callingCode` | string | 國碼 | BookingListCard, ReviewReserveCard |
| `mobilePhone` | string | 手機號碼 | BookingListCard, ReviewReserveCard |
| `email` | string | Email | ReviewReserveCard |
| `birthday` | string? | 生日 | 壽星判定、ReviewReserveCard |
| `isBlocklist` | boolean? | 黑名單 | 全部卡片（icon） |
| `isDiplomat` | boolean? | 外交禮遇 | 全部卡片（icon） |
| `crmId` | string? | CRM 會員 ID | ReviewReserveCard |
| `crmSourceSystem` | string? | CRM 來源系統 | CRM icon 判定（A7CRM/A6WS） |
| `cardNumber` | string? | 會員卡號 | BookingListCard (條件), ReviewReserveCard |
| `cardTypeCode` | string? | 卡片類型代碼 | ReviewReserveCard |
| `cardTypeName` | string? | 卡片類型名稱 | ReviewReserveCard |
| `saluteCode` | string | 稱謂代碼 | CreateReserveCard |
| `languageCode` | string | 語言代碼 | CreateReserveCard |

### 桌位資訊 (reservationDetailList)

陣列，每個元素代表一張桌子：

| 欄位 | 型別 | 說明 |
|------|------|------|
| `deskNos` | string | 桌號 |
| `onsiteStatus` | OnsiteStatus? | 該桌的現場狀態 |
| `seatingTime` | string? | 入座時間 |
| `persNum` | number | 該桌座位數 |

### 訂金資訊

| 欄位 | 型別 | 說明 |
|------|------|------|
| `isDepositRequired` | 'Y'/'N' | 是否需要訂金 |
| `depositStatus` | PDP/NDP | 訂金狀態（已付/未付） |
| `depositAmountDue` | number | 應付訂金金額 |
| `depositDueDatetime` | string? | 訂金截止時間 |
| `paymentOverdueAutoCancel` | 'Y'/'N'? | 逾期是否自動取消 |

### 候位資訊

| 欄位 | 型別 | 說明 |
|------|------|------|
| `expectedWaitTime` | number | 預計等待時間（分鐘） |
| `waitlistStartTime` | string? | 候位開始時間 |
| `waitlistGroup` | string? | 候位群組 |
| `waitlistNumber` | string? | 候位號碼 |
| `waitlistNotificationTime` | string? | 候位通知時間 |
| `waitlistSource` | ResSource? | 候位來源 |
| `canWaitingToReservation` | 'Y'/'N' | 可否從候補轉正 |

## 業務說明

訂位模組（FBM0100000）是 FBM 系統的核心，管理餐廳訂位的完整生命週期：建立、檢視、通知、入座、離席、取消與復位。

### 視圖模式

系統提供四種視圖，使用者可切換：

| 視圖 | 說明 | 主要元件 |
|------|------|----------|
| 列表 (List) | 訂位清單 + 側邊詳情 | SidebarList → BookingListCard |
| 時間軸 (Timeline) | 依時間橫向排列 | TimelineViewCard |
| 桌圖 (Diagram) | Canvas 桌位圖 | Fabric.js 繪圖 |
| 現場客/候位 | 現場候位管理 | SidebarList → BookingListCard |

所有視圖的側邊欄都使用 **BookingListCard** 顯示摘要，點擊後開啟 **ReviewReserveCard** 詳情。

### 訂位卡片系統

訂位卡是整個模組的核心 UI，由六個元件組成：

| 元件 | 角色 | 複雜度 | 文件 |
|------|------|--------|------|
| [[BookingListCard]] | 側邊欄摘要卡 | 中 | done |
| [[ReviewReserveCard]] | 完整詳情檢視 | 高 | todo |
| [[CreateReserveCard]] | 建立/編輯表單 | 極高 | todo |
| TimelineViewCard | 時間軸條狀卡片 | 中 | todo |
| ListViewCard | 列表模式卡片 | 中 | todo |
| PreviewReserveCard | 載入骨架 | 低 | — |

### 訂位生命週期

```
建立 (Create) → 確認 (CNF) → 通知 (NTF) → 入座 (STD) → 離席 (LFT)
                    │                            ↑
                    ├── 取消 (CXL) ──復位──→ 確認 (CNF)
                    │
                    └── 候補 (WLB) ──轉正(WTC)──→ 確認 (CNF)
```

### 四維狀態系統

訂位的顯示行為由四個獨立狀態維度交叉決定：

| 維度 | 欄位 | 值 | 說明 |
|------|------|----|------|
| 訂位類型 | `reservationType` | RSV / WLB / WIO / WLK | 訂位、候補、候位、現場客 |
| 訂位狀態 | `reservationStatus` | CNF / CXL / WTC / WIO / WLB | 確認、取消、候補轉正... |
| 現場狀態 | `onsiteStatus` | NST / STD / LFT / CXL | 未入座、已入座、已離席、已取消 |
| 通知狀態 | `reminderStatus` | NNF / NDF / NTF / HLD | 未通知、失敗、已通知、客人保留 |

### 訂位類型

| Code | 名稱 | 說明 |
|------|------|------|
| RSV | 訂位 | 一般預約訂位 |
| WLB | 候補 | 訂位候補，可轉正 |
| WIO | 候位 | 現場排隊候位 |
| WLK | 現場客 | 未預約直接到場 |

## 重要邏輯

### 遲到判定

顯示「遲到」的條件（全部同時成立）：
1. `reservationType != WLB`（候補不算遲到）
2. `startTime` 有值
3. 有指派桌位（`reservationDetailList.length > 0`）
4. 所有桌位的 `onsiteStatus` 都是 `NST`（還沒入座）
5. 現在時間 - `startTime` > `holdTime` 分鐘

### 狀態圖示優先序

StatusIconRender 的判定有嚴格的優先順序，詳見 [[狀態系統]]。

### 功能開關

| Config Key | 影響 |
|---|---|
| `enableWaitlistFeature` | 是否啟用候位/候補功能 |
| `showMemberCategory` | 是否顯示 CRM 等級徽章 |
| `showMemberCardNumber` | 是否顯示會員卡號 |
| `showMemberBirthday` | 是否顯示壽星標記 |
| `reservationHoldTime` | 遲到計算的保留時間（分鐘） |

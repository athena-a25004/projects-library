# 命名規範

## 檔案與目錄

| 類型 | 規範 | 範例 |
|------|------|------|
| Vue 元件 | PascalCase | `ReservationCard.vue` |
| Composable | camelCase + use 前綴 | `useReservation.ts` |
| 工具函式 | camelCase | `formatDate.ts` |
| API 模組 | camelCase | `reservationApi.ts` |
| Store | camelCase + use 前綴 | `useReservationStore.ts` |
| 路由目錄（Nuxt） | kebab-case | `reservation-list/` |

## 變數與函式

| 類型 | 規範 |
|------|------|
| 響應式資料（ref/reactive） | camelCase |
| 常數 | UPPER_SNAKE_CASE |
| 型別/介面 | PascalCase，介面加 `I` 前綴（待確認團隊慣例） |
| Emit 事件 | kebab-case |
